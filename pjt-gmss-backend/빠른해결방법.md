# 빠른 해결 방법

## ❌ 현재 에러
```
Error: Could not find or load main class com.study.spring.Application
Caused by: java.lang.ClassNotFoundException: com.study.spring.Application
```

## 🔧 원인
Eclipse/STS가 프로젝트를 제대로 빌드하지 못했거나, 빌드 경로가 잘못 설정되어 있습니다.

---

## ✅ 해결 방법 (순서대로 시도)

### 방법 1: Eclipse/STS 프로젝트 완전 리빌드 ⭐ (가장 효과적)

1. **프로젝트 Clean**
   - Eclipse/STS에서 **Project** 메뉴 → **Clean...**
   - backend 프로젝트 선택
   - "Clean all projects" 체크
   - OK 클릭

2. **Gradle 프로젝트 Refresh**
   - **backend 프로젝트 우클릭** → **Gradle** → **Refresh Gradle Project**

3. **프로젝트 Refresh**
   - **backend 프로젝트 우클릭** → **Refresh** (F5)

4. **Build Automatically 확인**
   - **Project** 메뉴 → **Build Automatically** 체크 확인

5. **서버 실행**
   - `Application.java` 파일 우클릭
   - **Run As** → **Java Application** 또는 **Spring Boot App**

---

### 방법 2: 빌드 경로 확인 및 수정

1. **프로젝트 우클릭** → **Properties**
2. **Java Build Path** 선택
3. **Source** 탭에서 다음 경로가 있는지 확인:
   - `src/main/java`
   - `src/main/resources`
   - `src/test/java`
4. 없다면 **Add Folder...** 버튼으로 추가
5. OK → 프로젝트 Refresh

---

### 방법 3: Gradle로 직접 빌드 (터미널)

**PowerShell 또는 CMD에서:**

```powershell
# 1. backend 폴더로 이동
cd C:\KSJ\Fiveguys\pjt-gmss\backend

# 2. Gradle 클린 빌드
.\gradlew clean build -x test

# 3. 빌드 성공 확인
# "BUILD SUCCESSFUL" 메시지가 나와야 합니다
```

**빌드 후:**
- Eclipse/STS 재시작
- backend 프로젝트 Refresh (F5)
- 서버 실행

---

### 방법 4: 워크스페이스 메타데이터 재생성

1. **Eclipse/STS 완전 종료**

2. **워크스페이스 폴더로 이동**
   - 보통: `C:\Users\사용자명\eclipse-workspace`

3. **.metadata 폴더 백업 (선택사항)**
   - `.metadata` 폴더를 `.metadata_backup`으로 이름 변경

4. **Eclipse/STS 재시작**

5. **프로젝트 다시 임포트**
   - **File** → **Import** → **Existing Gradle Project**
   - `C:\KSJ\Fiveguys\pjt-gmss\backend` 선택

---

### 방법 5: Run Configuration 재생성

1. **Run** 메뉴 → **Run Configurations...**

2. **Java Application** 또는 **Spring Boot App**에서 기존 실행 설정 삭제

3. **New Configuration** 생성:
   - Name: `GMSS Backend`
   - Project: `backend`
   - Main class: `com.study.spring.Application`

4. **Apply** → **Run**

---

## 🎯 가장 빠른 해결 (추천)

Eclipse/STS에서:

1. **Project** → **Clean...** → backend 선택 → OK
2. **backend 프로젝트 우클릭** → **Gradle** → **Refresh Gradle Project**
3. **F5** (Refresh)
4. **Application.java** 우클릭 → **Run As** → **Spring Boot App**

이렇게 하면 보통 해결됩니다!
