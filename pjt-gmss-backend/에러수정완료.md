## ✅ 수정 완료 항목

### 1. ReviewService 타입 불일치 에러
**문제:** `CnslRepository.findById(Integer)`가 `Long` 타입을 기대함

**위치:** `ReviewService.java` 44번째 줄

**수정 내용:**
```java
// 수정 전
Cnsl_Reg cnsl = cnslRepository.findById(cnslId)

// 수정 후
Cnsl_Reg cnsl = cnslRepository.findById(cnslId.longValue())
```

**상태:** ✅ 수정 완료

---

### 2. BbsRepository 메서드 명시적 쿼리 추가
**문제:** Bbs 엔티티의 필드명이 `bbs_div`로 언더스코어 사용 (카멜케이스 아님)

**위치:** `BbsRepository.java`

**수정 내용:**
- `findByDelYnOrderByCreatedAtDesc` → `@Query` 사용
- `findByBbsDivAndDelYnOrderByCreatedAtDesc` → `@Query` 사용

**이유:** Spring Data JPA의 자동 메서드 생성은 카멜케이스를 기대하므로, 언더스코어 필드명은 명시적 쿼리 필요

**상태:** ✅ 수정 완료
